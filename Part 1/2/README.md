# Project 2: Raw React

Building a React application from scratch without complex build tools like Vite or Create React App.

## Key Features

- Manual setup of React and ReactDOM.
- Implementing a simple counter and UI logic.
- Understanding the transformation from JSX to React.createElement.

## Tech Stack

- React
- ReactDOM
- JavaScript (CDN)
