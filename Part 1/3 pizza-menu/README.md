# Project 3: Pizza Menu

A beautiful, static pizza menu application demonstrating component composition and state management basics.

## Key Features

- Reusable Pizza components.
- Conditional rendering for "Sold Out" status.
- Props-driven data flow.
- Custom CSS styling for a premium feel.

## Tech Stack

- React
- JavaScript
- CSS

## Setup

1. `npm install`
2. `npm start`
