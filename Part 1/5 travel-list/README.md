# Project 5: Travel List

A full-featured packing list application for travelers, focusing on list management and data persistence patterns.

## Key Features

- Adding, deleting and toggling items.
- Conditional rendering for item status.
- Interactive form with controlled elements.
- Stats calculation using derived state.
- Sorting functionality (by input order, description, or status).

## Tech Stack

- React
- JavaScript
- CSS

## Setup

1. `npm install`
2. `npm start`
