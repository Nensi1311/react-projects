# Project 6: Eat-N-Split

A split-the-bill application designed to manage balances between friends after dining together.

## Key Features

- Dynamic list of friends with real-time balance tracking.
- Interactive "Split Bill" form.
- Selection state to manage which friend is currently active.
- Calculations for complex splitting logic.

## Tech Stack

- React
- JavaScript
- CSS

## Setup

1. `npm install`
2. `npm start`
