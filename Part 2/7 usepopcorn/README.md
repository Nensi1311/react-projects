# Project 7: usePopcorn

An advanced movie database application that allows users to search for movies, rate them, and manage their personal watch list.

## Key Features

- Dynamic movie search via the OMDB API.
- Star rating system with a reusable component.
- Persistence of movie ratings and watch list.
- Deep dive into hooks: `useEffect` for API calls, `useRef` for DOM access and persistent values.
- Custom hooks for shared logic (localStorage, geolocation).

## Tech Stack

- React
- JavaScript
- CSS (Modules/Global)

## Setup

1. `npm install`
2. `npm start`
