# Project 8: How React Works

A deep-dive project into the internal mechanics of React, exploring the rendering lifecycle and reconciliation.

## Key Features

- Visualizing component rendering and re-rendering.
- Experimenting with keys and their impact on the DOM.
- Understanding the difference between state updates and UI rendering.
- Exploring the virtual DOM and the diffing algorithm.

## Tech Stack

- React
- JavaScript
- CSS

## Setup

1. `npm install`
2. `npm start`
