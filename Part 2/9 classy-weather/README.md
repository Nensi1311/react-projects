# Project 9: Classy Weather

A weather application built using legacy class-based components, comparing them with modern functional components.

## Key Features

- Managing state and lifecycle methods (`componentDidMount`, `componentDidUpdate`).
- Fetching weather data from an external API.
- Implementation of a search bar and results display using classes.
- Comparing class boilerplate with hook-based efficiency.

## Tech Stack

- React (Class Components)
- JavaScript
- CSS

## Setup

1. `npm install`
2. `npm start`
