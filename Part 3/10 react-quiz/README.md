# Project 10: React Quiz

A high-performance quiz application using advanced state management patterns to track progress and scoring.

## Key Features

- Complex state management with `useReducer`.
- Dynamic question loading and timer implementation.
- Real-time scoring and progress bar.
- Result screen with performance summary.

## Tech Stack

- React
- JavaScript
- CSS

## Setup

1. `npm install`
2. `npm run server` (in one terminal)
3. `npm start` (in another terminal)
