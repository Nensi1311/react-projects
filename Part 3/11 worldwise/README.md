# Project 11: WorldWise

A location-tracking "trip diary" application that uses routing and maps to help users remember where they've been.

## Key Features

- Advanced routing with `react-router`.
- Interactive maps using Leaflet.
- Context API for global state management.
- Dynamic data fetching and URL-based state.
- Forms for adding new cities based on map clicks.

## Tech Stack

- React
- React Router
- Leaflet (Maps)
- Context API
- CSS Modules

## Setup

1. `npm install`
2. `npm run dev`
