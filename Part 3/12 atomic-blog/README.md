# Project 12: Atomic Blog

A blog application focused on the Context API and performance optimization techniques for large-scale state.

## Key Features

- Implementation of a global state system with the Context API.
- Strategies for avoiding unnecessary re-renders.
- Dynamic search and filter across blog posts.

## Tech Stack

- React
- Context API
- JavaScript
- CSS

## Setup

1. `npm install`
2. `npm start`
