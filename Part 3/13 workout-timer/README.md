# Project 13: Workout Timer

A precision workout timer focusing on memoization and high-frequency state updates.

## Key Features

- Real-time countdown and interval management.
- Performance optimization using `useMemo` and `useCallback`.
- Memo-wrapped components to minimize render cycles.
- Dynamic styling based on timer status.

## Tech Stack

- React
- JavaScript
- CSS

## Setup

1. `npm install`
2. `npm start`
