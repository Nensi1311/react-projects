# Project 14: Redux Intro

An introduction to managing global application state using Redux, from classic patterns to modern Redux Toolkit (RTK).

## Key Features

- Designing global stores and reducers.
- Implementing actions and dispatchers.
- Comparing "Classic Redux" with "Modern Redux Toolkit".
- Asynchronous actions using Thunks.

## Tech Stack

- React
- Redux
- Redux Toolkit (RTK)
- JavaScript

## Setup

1. `npm install`
2. `npm start`
