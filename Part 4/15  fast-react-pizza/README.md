# Project 15: Fast React Pizza Co.

A professional pizza ordering application featuring complex state, data loading patterns, and a sleek user interface.

## Key Features
- Advanced routing with React Router (Data Loaders and Actions).
- Global state management with Redux Toolkit (Cart, User, Order).
- Dynamic shopping cart with real-time updates.
- Checkout flow with order creation and verification.
- Geolocation for delivery address.

## Tech Stack
- React
- React Router (v6+)
- Redux Toolkit
- Tailwind CSS
- JavaScript

## Setup
1. `npm install`
2. `npm run dev`
