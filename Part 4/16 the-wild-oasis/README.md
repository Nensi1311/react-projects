# Project 16: The Wild Oasis (Staff App)

A professional internal management system used by hotel staff to manage bookings, cabins, and guests.

## Key Features

- Full CRUD operations for cabins and bookings.
- Advanced data fetching and caching with React Query.
- Authentication and authorization with Supabase.
- Interactive charts and dashboards for hotel stats.
- Dark mode and custom styling with Styled Components.
- Multi-step forms and professional data tables.

## Tech Stack

- React
- React Query
- Supabase (Auth, DB, Storage)
- Styled Components
- React Hook Form
- Recharts
- JavaScript

## Setup

1. `npm install`
2. `npm run dev`
