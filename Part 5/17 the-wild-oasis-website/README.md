# Project 17: The Wild Oasis (Customer Website)

A high-performance, SEO-friendly customer-facing website built with the modern Next.js App Router.

## Key Features

- Server-Side Rendering (SSR) and Static Site Generation (SSG).
- Dynamic routing and metadata for SEO optimization.
- Secure customer login via NextAuth (Google/Credentials).
- Complex reservation system with availability checks.
- Image optimization using the Next.js `Image` component.
- Modern UI with Tailwind CSS.

## Tech Stack

- Next.js (App Router)
- React Server Components (RSC)
- Supabase
- NextAuth.js
- Tailwind CSS
- JavaScript

## Setup

1. `npm install`
2. `npm run dev`
