# Project 18: Next.js Pages Router Restructuring

A transition project focused on refactoring and restructuring applications to work with the Next.js Pages Router architecture.

## Key Features

- Migrating from App Router to Pages Router.
- Implementing `getStaticProps` and `getServerSideProps` for data fetching.
- API route implementation for server-side logic (e.g., Contact Form).
- Global layout patterns in `_app.js`.
- SEO optimization using `next/head`.

## Tech Stack

- Next.js (Pages Router)
- React
- Tailwind CSS
- Supabase
- JavaScript

## Setup

1. `npm install`
2. `npm run dev`
