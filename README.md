# React Projects 🚀

This repository contains 18 projects completed during my comprehensive React development journey. The course covered everything from fundamental JavaScript and React basics to advanced topics like state management, performance optimization and full-stack development with Next.js and Supabase.

## Project Structure

### [Part 1: The Fundamentals](Part%201)

1.  **React Introduction** - Exploring the basics of React components and JSX.
2.  **Raw React** - Building components without build tools to understand the core engine.
3.  **Pizza Menu** - Creating a static menu with components, props, and styling.
4.  **Steps Component** - First encounter with state management and event handling.
5.  **Travel List** - A full CRUD application focusing on controlled elements and derived state.
6.  **Eat-N-Split** - Advanced state management with multiple sibling components.

### [Part 2: Intermediate React](Part%202)

7.  **usePopcorn** - Fetching data from APIs and using advanced hooks like `useEffect` and `useRef`.
8.  **How React Works** - In-depth look at rendering, diffing, and the virtual DOM.
9.  **Classy Weather** - Building a class-based weather application to understand legacy React.

### [Part 3: Advanced State Management](Part%203)

10. **React Quiz** - Using `useReducer` to manage complex application state.
11. **WorldWise** - Implementing routing with `react-router` and using the `Context API`.
12. **Atomic Blog** - Context API performance optimization and state patterns.
13. **Workout Timer** - Advanced memoization techniques (`useMemo`, `useCallback`).
14. **Redux Intro** - Moving state management to Redux (Legacy and Toolkit).

### [Part 4: Professional Development](Part%204)

15. **Fast React Pizza Co.** - Real-world application with complex routing, data loading, and Redux.
16. **The Wild Oasis (Staff App)** - Professional internal management system with React Query, styled-components, and Supabase.

### [Part 5: Full-Stack with Next.js](Part%205)

17. **The Wild Oasis (Customer Website)** - Modern full-stack website using the Next.js App Router and server components.
18. **Next.js Pages Router Restructuring** - Transitioning and restructuring projects for the Next.js Pages Router architecture.

## How to Run Any Project

1.  **Navigate** to the specific project folder:
    ```bash
    cd "Part X/Project-Name"
    ```
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Start Development Server**:
    - For most projects: `npm run dev`
    - For Project 14 (Redux Intro) and earlier: `npm start`
4.  **Open in Browser**: Navigate to `http://localhost:3000` or `http://localhost:5173` (depending on the project).
